Мой тестовый проект на GoLang.
Для настройки деплоя сначала надо будет арендовать и настроить сервер.
Подключение указать по SSH ключу и скопировать туда публичный ключ
Приватный ключ нужно будет сохранить в секретную переменную во вкладке Settings -> Secrets and variables -> Actions -> New repository Secrets
Назвать переменную нужно DEPLOY_SSH_KEY
Пароль от хоста уже добавлен в секретные переменные
После настройки сервера, нужно записать его IP в файл workwlows/deploy.yaml на строку HOST
